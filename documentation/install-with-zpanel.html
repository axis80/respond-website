Install with ZPanel

This tutorial will walk you through installing Respond on an Ubuntu server with ZPanel installed.  This tutorial is provided by Glenn and available on his Github page.  If you are using Digital Ocean, use Glen's referral code when you sign up to show support.  

Prerequisites

To get started, make sure you have ZPanel (or Sentora) installed and you are running Ubuntu 12.04 LTS.

1. Logon to your server as root

sudo -i

2. Ensure you are in the home directory

cd

3. Download the installer

wget https://raw.githubusercontent.com/glenskie16/Composer-install/master/composerinstall.sh

4. Make the installer executable

chmod +x composerinstall.sh

5. Run the installer

sudo ./composerinstall.sh

6. Follow the onscreen instructions during the installation
